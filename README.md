# grannev.github.io

This is a static personal website with a retro aesthetic inspired by Rebuild
the World and Unix Wizards.

## Properties

- XHTML 1.0 Strict and CSS
- No JavaScript
- No frameworks or build system
- No runtime dependencies
- Assets stored locally
- English and Russian pages
